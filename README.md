# Resolution Switcher WinForms

Простая программа для просмотра и изменения разрешения экрана в Windows. Переключает между тремя предустановленными разрешениями.

## Функциональность

- Показывает текущее разрешение экрана
- Переключение между разрешениями: 640x480 → 800x600 → 1920x1080
- Работа через WinAPI (user32.dll)
- Проверка поддерживаемости разрешения перед применением

## Интерфейс

- Кнопка "button1" — показать текущее разрешение
- Кнопка "button2" — циклическое переключение разрешений

## Куда можно встроить

- В утилиту для настройки мониторов
- В программу тестирования дисплеев
- В лаунчер с автонастройкой под игру

## Требования
- Windows
- .NET Framework / .NET Core с WinForms
- Права администратора (для изменения разрешения)

## Примечание

- Для изменения разрешения требуются права администратора. Программа циклически переключает разрешения при каждом нажатии кнопки.
